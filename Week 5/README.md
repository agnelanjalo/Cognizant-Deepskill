This repo contains all 10 hands-on exercises for Module 2, building a single running project — the **Student Portal** — progressively from plain HTML/CSS/JS through React, Angular, and Vue.js, finishing with accessibility and advanced state management.

Each folder has its own `README.md` with setup and run instructions — start there.

## Software used
- VS Code, Node.js (LTS) + npm, Git
- Chrome/Firefox with DevTools, Lighthouse, axe DevTools
- Vite (React, Vue), Angular CLI

## Notes
- Hands-On 1–4 are plain HTML/CSS/JS — open directly or serve with `npx serve .`
- Hands-On 5–10 are framework projects — each has its own `package.json`; run `npm install` inside that folder before `npm run dev` (or `ng serve` for Angular).
- `node_modules/` is intentionally excluded from every framework folder per the submission guidelines.
